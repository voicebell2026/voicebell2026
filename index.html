FAQ

자주 묻는 질문 Frequently asked questions

  1. 다운로드 버튼 클릭
  2. 압축 파일 다운로드 → 압축 풀기
  3. 바탕화면에 server 폴더와 start_voicebell.bat이 함께 있으면 준비 완료
  4. start_voicebell.bat 더블클릭
  5. 검은 창이 잠깐 뜨다가 Chrome이 자동으로 열림
  6. Windows 방화벽 허용 팝업 → 개인/공용 네트워크 둘 다 체크 → 액세스 허용 클릭 ⚠️
  7. "연결이 비공개" 페이지 → 고급 → localhost (으)로 이동 클릭
  8. 결제 시 사용한 이메일 입력 → 인증 완료 (인터넷 연결 필수)
  9. 카운터 화면 시작 🎉

자세한 내용은 FAQ를 참조하세요.

  1. Click the Download button
  2. Download the zip file → Extract
  3. Make sure server folder and start_voicebell.bat are both on your desktop
  4. Double-click start_voicebell.bat
  5. A black window appears briefly, then Chrome opens automatically
  6. Windows Firewall popup → Check both Private and Public networks → Click Allow access ⚠️
  7. "Your connection is not private" → Advanced → click localhost (unsafe)
  8. Enter the email used at checkout → Verified (Internet connection required)
  9. Counter screen starts 🎉

카운터 PC는 Windows 10 이상이 필요합니다. 키친·키오스크는 Chrome 브라우저가 설치된 기기라면 사용 가능합니다. iPad, Android 태블릿, Mac 등을 지원합니다. 단, 10인치 이상 태블릿을 권장하며 스마트폰처럼 화면이 작은 기기는 최적화된 화면 구현이 어려울 수 있습니다. The counter PC requires Windows 10 or later. For kitchen and kiosk, any device with Chrome works — iPad, Android tablet, Mac, and more. Note: a tablet with a 10-inch or larger screen is recommended. On smaller screens such as smartphones, the display may not be fully optimized.

모든 데이터는 사장님의 카운터 PC에만 저장됩니다. VoiceBell 서버로 전송되지 않습니다. 손님 개인정보는 수집하지 않습니다.

⚠️ 서버가 실행 중인 상태에서만 데이터가 저장됩니다. 영업 중에는 반드시 서버를 켜두세요.

  • 일간: 카드/현금 결제 처리 시 실시간 저장
  • 월말·분기말·연말: 해당 시점에 자동 생성 (서버 실행 중이어야 함)

All data is stored only on your counter PC. Nothing is sent to VoiceBell servers. No customer personal information is collected.

⚠️ Data is only saved while the server is running. Please keep the server on during business hours.

  • Daily: Saved in real time when card/cash payment is processed
  • Monthly/Quarterly/Annual: Auto-generated at the relevant closing time (server must be running)
